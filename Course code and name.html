Length: See detailed specifications below (for report submission, Title Page, Table of Content, References, and Appendix are not included in the length limit); using 1.5 line spacing, Times New Roman font size 12 is recommended

· CLO3: apply the knowledge of various ICT components to create a prototype of a product/service

· CLO4: identify principles of collaborative work and explain potential conflict resolution techniques

· CLO5: participate effectively and creatively in a team environment

· CLO6: prepare a professional and comprehensively written IT report within provided parameters
